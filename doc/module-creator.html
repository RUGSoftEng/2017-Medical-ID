Module: Creator

The Creator module power both document Creators. It handles all logic behind the form, including retrieval of data. To actually be able to generate documents, the Creator needs additional modules corresponding to those documents. These modules can use the fields() method of the Creator for input data. The Creator also needs to be provided with an endpoint for data persistency. Finally, Creator.init() must be called to start the Creator form, filling it with pre-set data.
